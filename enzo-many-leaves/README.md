# Enzo and the Many Leaves

A small interactive web story for mobile and laptop browsers. It has no dependencies, build step, tracking, or network calls.

## Preview locally

From this folder, run:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000` in a browser.

## Publish with GitHub Pages

Copy the entire `enzo-many-leaves` folder into the repository that already hosts your GitHub Pages site, commit it, and push. The story will be available beneath that site's URL at `/enzo-many-leaves/`.

All paths are relative, so the folder can be renamed before publishing if desired.
